# Project Title: Bad Morning

Bad Morning is a 2D top-down pixel-art zombie survival shooter developed in one week at the University of Colorado Colorado Springs. The game focuses on atmosphere, exploration, and survival, combining combat with environmental storytelling and interactive world elements.

## Play the Game
This project is not currently hosted online, but it is fully playable by downloading the provided ZIP file and running the included executable. No additional software or game engine is required to play.

## Features
- Top-down 2D zombie shooter gameplay
- Pixel-art visual style with atmospheric fog and vignette effects
- Dynamic health and stamina bars that follow the player
- Interactive objects including ammo pickups and story notes
- Zombie AI with randomly generated spawn points
- Music that dynamically fades based on zombie proximity
- Typed dialogue text effect for immersive storytelling
- Clear objective path and defined ending

## How to Play
Use the keyboard and mouse to survive and explore the map.

- Movement: WASD or arrow keys
- Shoot: Mouse click
- Interact: Click on interactive objects
- Objective: Explore the environment, manage resources, and follow story clues to reach the ending

## Installation (Player)
1. Download the ZIP file from the repository
2. Extract the contents to a folder
3. Run the `.exe` file to play

## Installation (Developer)
To modify or rebuild the project:
1. Install GameMaker Studio 2
2. Open the `.yyp` project file
3. Run or export the game from the IDE

## Technologies Used
- GameMaker Studio 2
- GML (GameMaker Language)
- Pixel art assets (original and collaborative)
